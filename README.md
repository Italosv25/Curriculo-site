#  Meu Currículo Online

Este é o repositório do meu site de currículo pessoal, desenvolvido para apresentar minha trajetória profissional, habilidades e projetos de forma acessível e direta.

---

##  Sobre o Projeto

O objetivo deste projeto foi criar uma versão web do meu currículo para facilitar o compartilhamento com recrutadores e manter uma presença online profissional. O site é focado em **clean design** e legibilidade.

###  Tecnologias Utilizadas

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização, layout (Flexbox/Grid) e responsividade.
* **Google Fonts:** Tipografia personalizada para melhor leitura.
